# Consegna:
Scrivi un programma che:
- Stampi i numeri da 1 a 100,
- Per i multipli di 3 stampi “Fizz” al posto del numero 
- Per i multipli di 5 stampi Buzz.
- Per i numeri che sono sia multipli di 3 che di 5 stampi FizzBuzz.
